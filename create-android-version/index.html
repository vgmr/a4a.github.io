Create android version

1 - Create Application in Google play store

Google Play Console

2 - Create upload keystore

# Run a shell on the android sdk container
docker-compose run --rm  -v $(pwd):/app -w:/app sdk bash

# create the key
keytool -genkey -v -keystore asm-vgmr.keystore -alias asm-vgmr-upload -keyalg RSA -keysize 2048 -validity 10000

# move the file in the correct folder and name it asm.keystore
cp <keyfile> /devops/assets/<bundle-name>/common/android/asm.keystore

3 - Update the gradle configuration

  • gradle.properties
  • proxy.configuration

The file build.gradle is customized here and will replace the default expo generated one. It needs to be aligned with the future expo version.
